# My-Cart
----------------
### A front end repo for ecommerce android application.

### Features - 
1. Authentication
2. Product Listing
3. Product Detailing 
  1. Thumbnail
  2. Images
  3. Overview
  4. Description
  5. Likes
  6. Profile Details/ Personalization
  7. Encrypted Server Communication.
